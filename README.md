# Operation Lean Premium — Web Version

This is the premium web-app version of your 70-day cut tracker.

## Included
- Premium iPhone-first interface
- No compulsory step target
- Flexible activity choices: bike, incline treadmill, StairMaster, rowing, football, padel, swimming or natural steps
- Daily missions, streaks, XP, levels and achievements
- Workout logging
- Calories, protein, water and sleep tracking
- Weekly waist and progress-photo check-ins
- Offline support after the first load
- Local storage on the device

## Notifications
Notifications are intentionally deferred to the next version.

## Install on iPhone
1. Upload all files to a static host such as Netlify, GitHub Pages, Cloudflare Pages or Vercel.
2. Open the hosted link in Safari.
3. Tap Share.
4. Tap Add to Home Screen.

## Privacy note
Entries are stored in Safari/localStorage on that device. Progress photos are also stored locally, so keep the originals in Photos. Clearing Safari website data can remove app data.
